## 2. Task A：3-α 温度敏感性指数（23分）

- 你实现的 `rate_3alpha(T)` 与 `sensitivity_nu(T0,h)` 核心思路：
严格按照恒星3-α反应率的温度依赖公式实现：q(T) = 5.09e11 * (T8 ** (-3.0)) * np.exp(-44.027 / T8)输入开尔文温度T，先计算归一化温度 T8，再代入公式计算反应率q(T)，直接返回结果;
温度敏感性指数定义为：v(T0) = (T0/q(T0)) * dq/dT
用前向差分近似导数 dq/dT：dq/dT = (q(T0 + h*T0) - q(T0)) / (h*T0)
（默认 h = 1e-8，平衡截断误差与舍入误差）
代入 T0 处的反应率 q(T0)，按定义计算 v(T0)。  
- 使用的温度点与步长：[1.0e8, 2.5e8, 5.0e8, 1.0e9, 2.5e9, 5.0e9], h = (1e-8)*T0  
- 计算结果表：

| T0 (K) | v(T0) |
|---:|---:|
| 1.0e8 | 41.03 |
| 2.5e8 | 14.61 |
| 5.0e8 | 5.81 |
| 1.0e9 | 1.40 |

- 物理解释：在低温与高温区，v 的变化说明了什么？
温度敏感性指数 v 的物理含义是：温度发生相对变化时，3-α反应率被放大的倍数，v 越大，系统对温度变化越敏感（越“点火敏感”）。
（1）低温区（T0 <= 1.0e8）
v 数值极大（约41），说明低温下3-α反应率对温度极其敏感：温度的微小升高会导致反应率呈指数级暴涨。
物理本质：3-α反应是强吸热的热核反应，低温下反应的能垒（库仑势垒）阻碍极强，反应率由指数项 e**(-44.027/T8) 主导，温度升高会大幅降低势垒穿透概率，因此反应率对温度变化极度敏感，是恒星“热核失控”（超新星爆发、氦闪）的核心物理原因。
（2）高温区（T0 >= 1.0e9）
v 显著降低，甚至变为负值（约-1~-2），说明高温下3-α反应率对温度的敏感性大幅下降，甚至随温度升高而降低。
物理本质：高温下库仑势垒的阻碍作用被大幅削弱，指数项的影响减弱，反应率由幂律项 T8**(-3) 主导：温度升高时，幂律项的衰减效应超过指数项的增强效应，导致反应率随温度升高反而下降，因此 v 变为负值，系统对温度变化不再敏感。
（3）整体趋势：
v 随温度升高，3-α反应率的敏感性会先增加，然后减少，这是由于指数项和幂律项的综合作用。
物理本质：指数项在低温下占主导，随温度升高而增加，随温度升高而减少。
幂律项在高温下占主导，随温度升高而减少，随温度升高而增加。
因此，3-α反应率的敏感性会先增加，然后减少。
